# Hardware & Embedded Systems Projects

An umbrella index of my Arduino / embedded-electronics work — hobby builds and final-year hardware projects. Each is a standalone repo, linked here as a submodule.

| Project | Description |
|---|---|
| [Arduino_codes](https://github.com/venkata-vazrala/Arduino_codes) | Raw Arduino sketch collection (all firmware referenced by the projects below) |
| [arduino-bluetooth-car](https://github.com/venkata-vazrala/arduino-bluetooth-car) | Bluetooth-controlled RC car on an RC rock-crawler chassis |
| [diy-fountain](https://github.com/venkata-vazrala/diy-fountain) | DIY tabletop fountain build |
| [private-alert-system](https://github.com/venkata-vazrala/private-alert-system) | GSM-based personal emergency alert device |
| [regenerative-braking-system](https://github.com/venkata-vazrala/regenerative-braking-system) | Final-year mini-project: regenerative braking prototype |
| [remote-communicator](https://github.com/venkata-vazrala/remote-communicator) | Long-range nRF24 communicator for rural connectivity |

## Clone with all submodules

```bash
git clone --recurse-submodules https://github.com/venkata-vazrala/hardware-embedded-projects.git
# or, if already cloned:
git submodule update --init --recursive
```
